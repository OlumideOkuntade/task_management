# Project Title: Task App

A simple Laravel application called Task built for managing tasks/projects

***Features***

* Create, read, update, and delete items

* Uses Laravel Eloquent ORM

* Clean MVC structure

* Responsive Blade UI

* Validation and error handling

***Tech Stack***

* Laravel 8 

* PHP 8+

* MySQL

* Composer

* Blade Templates

***Installation***
****Clone the repository****
git clone https://github.com/your-username/your-project.git
cd 

***Install dependencies***
* composer install


***Configure database***

Update .env with your database settings:

* DB_DATABASE=task
* DB_USERNAME=root
* DB_PASSWORD=

***Run migrations***
php artisan migrate

***Start development server***
php artisan serve
